The Basics of the Web and HTML

- The World Wide Web is a collection of HTML documents.
- HTML is Hypertext Mark-up Language
- HTML is the base for almost all pages on the web even if they have photos, video's, etc.
- Hyperlinks connect HTML pages together to form the web.
- The web was invented in the early 1990s and has about 30 billion pages today.

The Main Components that Make Up the Web

- You and me - Input commands/requests to the computer
- Computer with a web browser - Send requests to servers and display results
- Internet - Group of networks that links computers and servers
- Servers - Sends HTML documents to computers upon request

HTML Elements and Tags

- HTML Element Format - <Opening tag> Content <closing tag>
- The opening tag, content and closing tag are an element.
- The closing tag name is the same as the opening tag but with a / preceeding the name.
- Most elements have a closing tag but a few do not and are called empty elements.
- Tags cover such items as line return, text size/formating, and many others.
- Tags can have attributes like adding a url link to the text.
- Two types of tags, inline and block.
- Inline tags end the line; block tags make an invisible box around the text.
- Example tags are <em> which is emphasis, <br> is line return, <img> is image,and <a> is anchor

HTML Miscellaneous

- Current HTML version is Version 5
- The Head of an HTML page contains random stuff and metadata.
- The Body of an HTML page contains the viewable content.
- Computers only do what they are told to do.
